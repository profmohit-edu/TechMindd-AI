# Changelog

This project follows [Semantic Versioning](https://semver.org/).

## 1.0.1 - 2026-07-16

- Fixed automatic provider selection, manager-owned retry enforcement, and reflected quality reporting.
- Hardened package paths, template failures, DOCX ingestion, trace identifiers, metrics cardinality, uploads, and authenticated dashboard access.
- Fixed reverse-proxy URL generation, stable dependency resolution, and CPU-only container dependencies; removed obsolete templates/prompts and duplicate dependency declarations; and cleaned unused imports and circular plugin discovery.

## 1.0.0 - 2026-07-16

- Added workflow-driven generation, validation, quality, reflection, plugins, provider failover, REST API, and web dashboard.
- Added structured logging, Prometheus monitoring, security middleware, production containers, CI, and release documentation.
